## ğŸ“˜ 1.1 What is Pseudocode? (1 Hour)

### What is Pseudocode? (Very Simple Meaning)

Pseudocode is:

> **Writing the solution steps in simple English, but in a structured way**

It is:

* Not programming language
* Not English paragraph
* Somewhere in between

ğŸ’¡ Think of it like:

* Writing cooking steps before cooking
* Writing exam steps before solving maths

---

### Why Do Companies Care About Pseudocode?

In interviews:

* Companies test **logic**
* Not syntax
* Not memorization

Interviewers want to see:

* Can you think clearly?
* Can you explain your approach?

ğŸ‘‰ Pseudocode shows **clarity of thinking**

---

### Pseudocode Style Used in This Course

To avoid confusion, **one standard style** is used:

* `INPUT`
* `OUTPUT`
* `IF / ELSE`
* `FOR / WHILE`
* `FUNCTION`
* Proper indentation

Example:

```
INPUT number
IF number > 0
    OUTPUT "Positive"
ELSE
    OUTPUT "Not Positive"
END IF
```

ğŸ‘‰ Simple, readable, interview-friendly
